### Week 2 Objectives:
* Learn about Amazon Elastic Compute Cloud (EC2) and virtual machines.
* Understand key concepts: Instance Types, Amazon Machine Images (AMI), Key Pairs, and Elastic IPs.
* Learn Security Groups for firewall rules.
* Practice deploying a simple web application on an EC2 instance.
* Get introduced to high availability concepts: Elastic Load Balancing (ELB) and Auto Scaling.

### Week 2 Achievements:
* Gained deep understanding of AWS virtual machines (EC2) and how they scale.
* Created and launched an EC2 Instance using Amazon Linux AMI.
* Mastered network access security using Security Groups (opening ports 22, 80, 443).
* Successfully deployed Nginx web server and hosted a static website on EC2, accessible via public IP.
* Learned how to scale compute resources using ELB and Auto Scaling Groups for fault tolerance.
