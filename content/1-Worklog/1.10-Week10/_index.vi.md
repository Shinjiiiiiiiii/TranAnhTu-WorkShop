### Mục tiêu tuần 10:
* Tìm hiểu phương pháp luận DevOps và các dịch vụ CI/CD trên AWS: AWS CodePipeline, AWS CodeBuild và AWS CodeDeploy.
* Nghiên cứu 6 trụ cột cốt lõi của bộ khung kiến trúc chuẩn AWS Well-Architected Framework.
* Thực hành xây dựng quy trình CI/CD hoàn chỉnh để tự động hóa triển khai ứng dụng.
* Liên kết GitHub với AWS để tự động deploy ứng dụng khi đẩy code mới lên GitHub.
* Ôn tập tổng kết và đánh giá toàn bộ chương trình thực tập 10 tuần.

### Kết quả đạt được tuần 10:
* Nắm vững các khái niệm và thực tiễn về tích hợp liên tục và triển khai liên tục (CI/CD) trên môi trường Cloud.
* Hiểu cách đánh giá hệ thống dựa trên các trụ cột của AWS Well-Architected Framework để thiết kế hạ tầng tối ưu, bảo mật và hiệu quả chi phí.
* Xây dựng thành công đường ống triển khai tự động (Deployment Pipeline) bằng AWS CodePipeline.
* Kết nối thành công GitHub repository với AWS, cho phép tự động deploy code mới lên môi trường cloud mỗi khi thực hiện thao tác push code.
* Tổng kết, hệ thống hóa và nắm vững toàn bộ kiến thức về AWS và dịch vụ Cloud đã học trong suốt 10 tuần.
