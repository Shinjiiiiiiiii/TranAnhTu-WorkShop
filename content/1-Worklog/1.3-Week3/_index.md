### Week 3 Objectives:
* Learn about AWS storage options: Block Storage (EBS), Shared File Storage (EFS), Object Storage (S3), and Archival Storage (Glacier).
* Understand how to manage S3 Buckets and Objects.
* Practice hosting a Static Website on Amazon S3.
* Configure S3 features: Versioning, Lifecycle Policies, and Access Control Lists (ACLs).
* Understand data backup and restore mechanisms.

### Week 3 Achievements:
* Gained clear knowledge of when to use EBS, EFS, and S3 for different application needs.
* Created S3 Buckets and successfully managed object storage permissions.
* Configured and deployed a public-facing static website hosted purely on S3.
* Configured Object Versioning to track file changes and prevent accidental deletion.
* Implemented Lifecycle rules to automatically transition older, infrequently-accessed data to Amazon S3 Glacier to optimize storage costs.
