### Week 6 Objectives:
* Learn about Serverless computing on AWS using AWS Lambda.
* Study API routing using Amazon API Gateway.
* Understand messaging and event services: Amazon SNS, Amazon SQS, and Amazon EventBridge.
* Practice building a REST API backend with AWS Lambda.
* Integrate API Gateway with Lambda functions and test/monitor the API.

### Week 6 Achievements:
* Gained a clear understanding of Serverless design patterns and event-driven architectures.
* Developed serverless code inside Lambda functions using Python/NodeJS.
* Built and published a functional REST API backend using Amazon API Gateway.
* Successfully integrated API Gateway proxy endpoints to trigger Lambda functions.
* Mastered testing APIs using tools like Postman/cURL, and diagnosed serverless performance logs in Amazon CloudWatch.
