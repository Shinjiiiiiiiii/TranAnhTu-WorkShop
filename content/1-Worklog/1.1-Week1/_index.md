### Week 1 Objectives:
* Get introduced to Cloud Computing and the Amazon Web Services (AWS) ecosystem.
* Learn the differences between IaaS, PaaS, and SaaS cloud delivery models.
* Get familiar with the AWS Management Console interface.
* Create a secure AWS Account and set up IAM users, Multi-Factor Authentication (MFA), and Billing Alarms to prevent unexpected costs.

| Wed | - Learn Cloud Service Models: IaaS vs PaaS vs SaaS <br> - Create an AWS Free Tier Account | 05/06/2026 | 05/06/2026 | [AWS Cloud Service Models](https://docs.aws.amazon.com/whitepapers/latest/aws-overview/types-of-cloud-computing.html) |

### Week 1 Achievements:
* Gained a solid understanding of Cloud Computing concepts and AWS global infrastructure.
* Created a secure AWS Free Tier account.
* Successfully set up IAM users and groups, enforcing the principle of least privilege, and enabled MFA for root and IAM accounts.
* Created a CloudWatch Billing Alarm to monitor and alert when usage charges exceed $5.
* Acquired basic knowledge on how to navigate the AWS Console and manage user permissions.
